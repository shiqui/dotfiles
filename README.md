# Homebrew
I use [Homebrew](https://brew.sh/) to manage most of my tools.  
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
Then install from Brewfile

```
brew bundle
```
